# _templates/

Starter page structures for each entity type. Claude Code references these when creating new pages so structure and frontmatter stay consistent.

| Template | Used by Claude when creating |
|---|---|
| `concept.md` | a new concept page in `wiki/concepts/` |
| `vendor.md` | a new vendor page in `wiki/vendors/` |
| `sector.md` | a new sector page in `wiki/sectors/` |
| `source.md` | a new source page in `wiki/sources/` |
| `person.md` | a new person page in `wiki/people/` |
| `firm.md` | a new firm page in `wiki/firms/` |

Iterate the templates as you learn what works. The schema in `CLAUDE.md` references the entity types here; keep them consistent.
